# Mina Snapp: Mui Snapp Template

This template uses TypeScript.

## How to build

```sh
npm run build
```

## How to Serve

```sh
npm run serve
```

## How to run tests

```sh
npm run test
npm run testw # watch mode
```

## How to run coverage

```sh
npm run coverage
```
